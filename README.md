# SCARA Robot

A 4-DOF SCARA (Selective Compliance Assembly Robot Arm) developed as part of the MEP-1001: Manufacturing for Product Innovation course at IIT Delhi under the guidance of Prof. Pulak Mohan Pandey. The project involved the complete design, fabrication, electronics integration, and programming of a robotic arm capable of precise planar pick-and-place operations with an automated color-based sorting mechanism.

## Project Information

- **Course:** MEP-1001 – Manufacturing for Product Innovation
- **Institution:** IIT Delhi
- **Instructor:** Prof. Pulak Mohan Pandey
- **Duration:** March 2026 – April 2026
- **Team Size:** 6

## Features
- 4-DOF SCARA robotic manipulator
- Arduino-based motion control
- Inverse kinematics for end-effector positioning
- Stepper motor control using A4988 drivers
- Lead-screw based Z-axis mechanism
- Servo-actuated gripper
- Processing-based graphical user interface (GUI)
- RGB color sensor integration for automatic object classification and sorting

## Technologies Used
- Arduino
- Processing
- C/C++
- Stepper Motors (NEMA-17)
- Servo Motor
- A4988 Motor Drivers
- Lead Screw Mechanism
- RGB Color Sensor (TCS34725)


## Project Highlights
- Designed and fabricated the complete mechanical assembly.
- Implemented inverse kinematics for Cartesian positioning.
- Developed Arduino firmware for coordinated motor control.
- Built a Processing GUI for robot operation.
- Integrated an RGB color sensor to enable autonomous color-based object sorting, transforming the robot from an open-loop manipulator into a closed-loop decision-making system.
